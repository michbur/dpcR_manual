Rename object

Usage

rename_dpcr(x, exper = NULL, replicate = NULL)

Arguments

x
an adpcr or ddpcr object.
exper
a factor of new experiments' names. If NULL, experiments' names are not changed.
replicate
a factor of new replicates' ids. If NULL, replicates' names are not changed.

Description

Renames objects of class adpcr or ddpcr.
