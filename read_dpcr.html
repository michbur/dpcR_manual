Read digital PCR data

Usage

read_dpcr(input, format, ...)

Arguments

input
name of the input file (character) or input object (data.frame).
format
of the file, for example: "raw", "QX100", "BioMark".
...
additional parameters for the appropriate function. For example, if format has value "raw", the additional parameter must be adpcr.

Value

Always an object of adpcr or ddpcr type.

Description

Reads digital PCR data in various formats.

Details

Input files may be in .csv, .xls or .xlsx format. In case of Excel files with multiple sheets, only the first sheet will be analyzed.

See also

Read raw format files: read_raw. Read BioMark format files: read_BioMark. Read QX100 format files: read_QX100.
